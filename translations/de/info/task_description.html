Binary search ist eine effektive Methode, die die Divide-and-Conquer-Strategie einsetzt, um schnell ein Element in einer geordneten Liste zu finden, wobei der Suchraum bei jedem Schritt um die Hälfte reduziert wird. Diese Technik ist nicht nur ein Beispiel für die Effizienz von Algorithmen bei der Lösung von Suchproblemen, sondern zeigt auch, wie wichtig es ist, grundlegende Konzepte der Informatik zu verstehen, wie z.B. Divide-and-Conquer-Algorithmen, die für die Optimierung und Lösung komplexer Probleme auf intuitive Weise entscheidend sind.

Ziel dieser Übung ist es, die binäre Suche so zu implementieren, dass sie den Index des gesuchten Elements in der Liste (oder -1, wenn es nicht vorhanden ist) und die Anzahl der durchgeführten Schritte zurückgibt und so einen klaren Blick auf den Divide-and-Conquer-Prozess in Aktion bietet.
